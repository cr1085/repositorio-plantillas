¡Bienvenido a 3CB CLI!

Empieza a gestionar con facilidad y accede a tus plantillas favoritas.

Paso 1: Instalar el CLI

Para instalar el CLI globalmente, abre tu terminal y ejecuta el siguiente comando:

npm install -g 3cb-cli

Esto hará que el comando 3cb esté disponible en tu sistema.

 Paso 2: Iniciar Sesión

Para acceder a plantillas premium, primero inicia sesión en tu cuenta:

3cb login

Se te pedirá que ingreses tu correo electrónico y contraseña.

 Paso 3: Ver Plantillas

Para listar las plantillas gratuitas disponibles, ejecuta:

3cb free

Si ya iniciaste sesión, también podrás acceder a las plantillas premium:

3cb premium
 Paso 4: Descargar una Plantilla

Para descargar una plantilla específica, ejecuta el comando:

3cb download "Nombre de la Plantilla"

Por ejemplo, para descargar la "Plantilla Gratuita 1", usa:

3cb download "Plantilla Gratuita 1"

¿Tienes problemas? Consulta nuestra documentación oficial o contáctanos.
